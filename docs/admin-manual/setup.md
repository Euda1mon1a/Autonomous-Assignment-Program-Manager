# System Setup

Initial system setup and deployment guide.

---

## Deployment Options

### Docker Deployment (Recommended)

See [Getting Started](../getting-started/installation.md) for Docker setup.

### Production Deployment

For production environments:

1. Configure production `.env` file
2. Set up SSL/TLS certificates
3. Configure reverse proxy (Nginx)
4. Set up database backups
5. Configure monitoring

---

## Initial Configuration

After deployment:

1. Create admin account
2. Configure academic year
3. Import personnel data
4. Set up rotation templates
5. Configure notification settings

See [Quick Start](../getting-started/quickstart.md) for step-by-step instructions.
