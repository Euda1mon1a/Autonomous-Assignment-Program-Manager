***REMOVED******REMOVED*** User Roles

| Role | Description | Permissions |
|------|-------------|-------------|
| **Admin** | System administrator | Full access |
| **Coordinator** | Program coordinator | Schedule management |
| **Faculty** | Faculty member | View, approve swaps |
| **Resident** | Resident | View own schedule, swaps |
| **Clinical Staff** | Support staff | Read-only |
| **RN** | Registered nurse | Read-only |
| **LPN** | Licensed practical nurse | Read-only |
| **MSA** | Medical support assistant | Read-only |

***REMOVED******REMOVED*** Managing Users

***REMOVED******REMOVED******REMOVED*** Create User

1. Go to **Settings** → **Users**
2. Click **Add User**
3. Enter details and assign role
4. Click **Save**

***REMOVED******REMOVED******REMOVED*** Modify Permissions

1. Select user
2. Click **Edit**
3. Update role or permissions
4. Click **Save**

***REMOVED******REMOVED******REMOVED*** Deactivate User

1. Select user
2. Click **Deactivate**
3. User can no longer log in
