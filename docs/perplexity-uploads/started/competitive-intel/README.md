# Competitive Intelligence Upload

No source files needed — this is a pure web research session.

## Usage

1. No file uploads required
2. Paste the contents of `PROMPT.md` as the Perplexity Computer session prompt
3. Perplexity Computer will research 10+ scheduling platforms and compile findings

## Sections

| # | Section | Focus |
|---|---------|-------|
| 1 | Major Players | Deep dive on Amion, QGenda, MedHub, New Innovations, TigerConnect, etc. |
| 2 | Feature Matrix | Side-by-side comparison across 25+ feature categories |
| 3 | User Pain Points | Reddit, SDN, review sites — what do users actually hate? |
| 4 | Military/DoD Specific | Which tools do military programs use? DHA procurement requirements? |
| 5 | Whitespace & Opportunity | What's missing from the market? Where can we differentiate? |
| 6 | Market Sizing | TAM, military subset, contract values, growth trends |
