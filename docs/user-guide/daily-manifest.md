# Daily Manifest

The Daily Manifest provides a real-time view of "who is where" for any given day.

---

## Overview

Navigate to **Manifest** from the main navigation to see today's assignments grouped by location. This is the go-to view for coordinators to know where everyone is at any moment.

---

## Main Features

### Location-Based View

Assignments are grouped by location, making it easy to see:

- Who is at each clinic or hospital
- Current staffing levels vs. requirements
- Gaps in coverage

### Time Filtering

Toggle between time periods:

| Filter | Shows |
|--------|-------|
| **All Day** | Complete day assignments |
| **AM** | Morning shift only |
| **PM** | Afternoon shift only |

### Search

Use the search bar to quickly find a specific person and see their assignment for the day.

---

## Reading the Manifest

### Location Cards

Each location shows:

- **Location Name** - Clinic, hospital, or department
- **Staff Count** - Current / Required staffing
- **Status Indicator** - Green (adequate), Yellow (minimal), Red (understaffed)
- **Assigned Personnel** - List of people at this location

### Personnel Details

Each person entry shows:

- Name and role (Resident/Faculty)
- PGY level (for residents)
- Shift time (AM/PM/All)
- Assignment notes

---

## Staffing Summary

The top of the page shows overall staffing:

| Metric | Description |
|--------|-------------|
| **Total Assigned** | People with assignments today |
| **Coverage Rate** | Percentage of required positions filled |
| **Gaps** | Number of unfilled required positions |
| **Absences** | People on leave today |

---

## Navigation

### Date Selection

- Use the date picker to view any day
- **Today** button returns to current date
- Arrow buttons navigate day-by-day

### Quick Links

Click any person to:
- View their full schedule
- See their contact information
- Access their profile

Click any location to:
- View detailed location schedule
- See weekly coverage
- Access location settings

---

## Use Cases

### Morning Briefing

1. Open Daily Manifest at start of day
2. Review all location assignments
3. Identify any coverage gaps
4. Verify faculty supervision in place

### Coverage Emergencies

1. Filter to find gaps (red indicators)
2. Search for available personnel
3. Use swap system to fill gaps

### Shift Handoff

1. Review PM assignments
2. Confirm all positions covered
3. Note any special assignments

---

## Best Practices

!!! tip "Start Your Day Here"
    Make the Daily Manifest your first stop each morning to ensure
    all locations are properly staffed.

!!! tip "Watch the Colors"
    - Green = Good coverage
    - Yellow = Minimum staffing
    - Red = Understaffed - action needed

!!! tip "Check Absences"
    Review the absence count to understand any last-minute changes.
