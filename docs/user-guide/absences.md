## Overview

The Absences feature helps you track and manage all types of time off for residents and faculty, including vacation, sick leave, military deployment, conferences, and more.

### Why Track Absences?

- **Accurate Scheduling:** Ensure people aren't scheduled when they're unavailable
- **Coverage Planning:** Identify gaps and plan coverage in advance
- **Compliance:** Track required time off and vacation usage
- **Transparency:** Give everyone visibility into who will be out and when
- **Records:** Maintain documentation for administrative purposes

### Who Can Manage Absences?

**Everyone can:**
- View the absence calendar
- See their own absences
- Request time off

**Coordinators and Admins can:**
- Add absences for anyone
- Edit existing absences
- Delete absences
- Export absence data

## Viewing the Absence Calendar

### Accessing the Absence Page

1. Click **"Absences"** in the main navigation
2. You'll see the Absence Management page

The Absences page is your central hub for all time-off management.

### Page Header

At the top of the page, you'll see:
- **Page Title:** "Absence Management"
- **Description:** "Manage vacation, sick days, and other absences"
- **Export Button:** To download absence data
- **Add Absence Button:** To create new absence requests

### View Mode Toggle

Choose how you want to view absences:

#### Calendar View
- Visual monthly calendar
- Absences displayed as colored blocks on dates
- Easy to see overlapping absences
- Best for planning and overview

#### List View
- Tabular list of all absences
- Shows all details in a sortable table
- Easy to scan many entries
- Best for reviewing specific absences

**Switching views:**
1. Look for the view toggle buttons near the top
2. Click **"Calendar"** icon for calendar view
3. Click **"List"** icon for list view

### Type Filter

Filter absences by type to focus on specific categories:

**Filter Options:**
- All Types (default - shows everything)
- Vacation
- Medical (Sick days)
- Conference
- Personal / Family Emergency
- Deployment
- TDY (Temporary Duty)

**To filter:**
1. Locate the filter dropdown
2. Select the absence type you want to view
3. The view updates automatically

### Calendar View Details

The calendar view shows:

#### Month Navigation
- Previous/Next month arrows
- Current month and year display
- Ability to jump to specific months

#### Calendar Grid
- Standard month view (weeks as rows, days as columns)
- Absences appear as colored bars on their dates
- Multiple absences on one day stack vertically
- Hover over an absence to see details

#### Color Coding
Each absence type has a distinct color:
- **Vacation:** Blue
- **Medical:** Red
- **Deployment:** Orange
- **TDY:** Yellow
- **Conference:** Green
- **Personal/Family:** Purple

#### Absence Information
Clicking an absence shows:
- Person's name
- Absence type
- Start and end dates
- Number of days
- Notes (if any)

### List View Details

The list view displays a table with columns:

| Column | Information |
|--------|-------------|
| Person | Name of the person who will be absent |
| Type | Absence category (vacation, medical, etc.) |
| Start Date | First day of absence |
| End Date | Last day of absence |
| Days | Number of days total |
| Notes | Additional information |
| Actions | Edit/Delete buttons |

**Sorting:**
Click column headers to sort (if available in your version)

**Scrolling:**
Use your mouse wheel or scrollbar to view more entries

## Requesting Time Off

### Adding a New Absence

#### Step 1: Open the Add Absence Form

1. Click the **"Add Absence"** button (with plus icon)
2. The "Add Absence" dialog box will appear

#### Step 2: Select the Person

**If you're a coordinator/admin:**
- Choose the person from the dropdown list
- List shows all residents and faculty
- Use search/type to find names quickly

**If you're a resident:**
- The person field may be pre-selected to your name

#### Step 3: Choose Absence Type

Select the appropriate category from the dropdown:
- Vacation
- Deployment
- TDY
- Medical
- Family Emergency
- Conference

**See [Absence Types Explained](#absence-types-explained)** for help choosing.

#### Step 4: Set Dates

**Start Date:**
1. Click the start date field
2. A calendar picker will appear
3. Click the first day of your absence
4. The date will populate in the field

**End Date:**
1. Click the end date field
2. Use the calendar picker
3. Click the last day of your absence
4. End date must be equal to or after start date

**Date Tips:**
- Both dates are inclusive (counts both start and end day)
- For a single day off, set start and end to the same date
- Use the calendar navigation to change months
- You can also type dates directly (format: YYYY-MM-DD)

#### Step 5: Add Notes (Optional)

Use the notes field to provide additional information:
- Reason for absence
- Contact information
- Special instructions
- Coverage arrangements

**Examples:**
- "Family wedding in Chicago"
- "Attending conference in San Diego"
- "Will have limited cell service"
- "Dr. Smith covering my patients"

Notes are visible to coordinators and may be visible to others depending on your organization's settings.

#### Step 6: Submit the Request

1. Review all information for accuracy
2. Click **"Add Absence"** or **"Submit"** button
3. If successful, you'll see a confirmation message
4. The dialog will close
5. The absence now appears in the calendar/list

### Form Validation

The form will prevent you from submitting if:
- No person is selected
- No absence type is selected
- Start date is missing
- End date is missing
- End date is before start date

Error messages will appear in red text explaining what needs to be fixed.

### After Submitting

Once submitted:
- The absence is saved immediately
- It appears in the calendar and list views
- It will be considered during schedule generation
- You can edit or delete it if needed

## Absence Types Explained

Understanding the different absence types helps ensure proper tracking and reporting.

### Vacation

**What it is:**
- Planned annual leave
- Earned vacation time
- Personal time off

**When to use:**
- Holiday travel
- Family visits
- Personal trips
- Planned rest periods

**Planning notes:**
- Request well in advance
- Check with coordinators about timing
- Consider rotation schedules
- Coordinate with fellow residents to avoid gaps

### Medical (Sick Leave)

**What it is:**
- Illness or injury
- Medical appointments
- Recovery time
- Health-related absences

**When to use:**
- You're sick and can't work
- Scheduled medical procedures
- Doctor appointments
- Family member medical care

**Important:**
- May require medical documentation
- Coordinate with chief resident or program director
- Arrange coverage for critical responsibilities
- Update as your situation changes

### Deployment

**What it is:**
- Military active duty assignment
- Typically longer duration
- Often overseas or away from home station
- Mandatory military service

**When to use:**
- Active military deployment orders
- Extended military training
- Combat zone assignments
- Peacekeeping missions

**Planning notes:**
- Provide orders as soon as received
- Discuss impact with program director
- May affect training timeline
- Coordinate with military and program

### TDY (Temporary Duty)

**What it is:**
- Temporary duty assignment
- Short-term military assignment
- Usually days to weeks, not months
- Travel to another location for training or work

**When to use:**
- Military temporary duty orders
- Short training assignments
- Temporary transfer to another facility
- Military exercises

**Differences from deployment:**
- Shorter duration
- Less disruptive to training
- Often planned in advance

### Conference

**What it is:**
- Medical conferences
- CME (Continuing Medical Education) events
- Professional development
- Academic meetings
- Training workshops

**When to use:**
- Attending national/regional conferences
- Presenting research
- Required CME activities
- Specialty society meetings

**Planning notes:**
- May be program-approved education time
- Keep records for CME credits
- Provide conference information
- May need to present what you learned

### Family Emergency

**What it is:**
- Unexpected personal or family situations
- Emergencies requiring immediate attention
- Personal matters requiring time off

**When to use:**
- Family member illness or death
- Personal emergencies
- Urgent personal matters
- Situations requiring immediate absence

**Important:**
- These are typically unplanned
- Contact supervisor immediately
- May require documentation
- Keep program informed of timeline

## Editing Absences

### When to Edit

Edit an absence when:
- Dates change
- Wrong dates were entered initially
- Absence type needs correction
- Need to add or update notes
- Plans have changed

### How to Edit an Absence

#### From List View:
1. Find the absence in the list
2. Click the **"Edit"** button next to it
3. The "Edit Absence" dialog opens

#### From Calendar View:
1. Click on the absence block
2. Select "Edit" option (if available)
3. The "Edit Absence" dialog opens

### Edit Absence Dialog

The edit form shows all current information:
- Person (usually can't be changed)
- Absence type (can be changed)
- Start date (can be changed)
- End date (can be changed)
- Notes (can be changed)

**To make changes:**
1. Modify any fields that need updating
2. Click **"Save Changes"**
3. Confirm you want to save
4. The absence is updated immediately

### Who Can Edit

- **Your own absences:** Usually yes
- **Others' absences:** Only if you're a coordinator or admin
- Depends on your role and organization policy

## Deleting Absences

### When to Delete

Delete an absence when:
- Plans are cancelled
- Absence was entered in error
- Person is no longer taking time off
- Duplicate entry

### How to Delete an Absence

#### From List View:
1. Find the absence
2. Click the **"Delete"** button (red)
3. A confirmation dialog appears

#### From Calendar View:
1. Click on the absence
2. Click delete option
3. Confirmation dialog appears

### Confirmation Dialog

Before deleting, you'll see:
- "Delete Absence" title
- Warning: "Are you sure you want to delete this absence record? This action cannot be undone."
- Person's name and absence details
- **Cancel** button (gray) - keeps the absence
- **Delete** button (red) - permanently removes it

**Important:** Deletion is permanent and cannot be undone!

### After Deletion

Once deleted:
- Absence is removed immediately
- No longer appears in calendar or list
- Will not affect future schedule generation
- Cannot be recovered

**Tip:** If you might need the information later, consider editing the dates to move it to the past instead of deleting.

## How Absences Affect Scheduling

### Integration with Schedule Generation

When you generate a schedule, the system:

1. **Checks all absences** in the date range
2. **Prevents scheduling** people during their absences
3. **Considers coverage needs** created by absences
4. **Adjusts workload** for remaining staff

### Before Schedule Generation

**Best practices:**
1. Enter ALL known absences before generating
2. Review the absence calendar for conflicts
3. Check for adequate coverage during absences
4. Coordinate overlapping absences

**Why this matters:**
- Algorithm can't schedule absent people
- Helps avoid coverage gaps
- Reduces need for schedule regeneration
- Prevents compliance violations

### After Schedule Generation

If you add an absence after generating a schedule:
- The schedule may now have gaps
- Compliance violations may appear
- You'll need to regenerate or manually adjust

**Recommendation:** Always enter absences BEFORE generating.

### Coverage Planning

Use absences to plan coverage:

**Review upcoming absences:**
1. Check the Dashboard "Upcoming Absences" card
2. Review the absence calendar
3. Identify periods with multiple people out
4. Plan extra coverage or adjust schedules

**High-absence periods:**
- Holidays (Thanksgiving, Christmas, etc.)
- Summer vacation season
- Conference season
- End of academic year

### Impact on Compliance

Absences affect ACGME compliance:

**Positive impacts:**
- Vacation days count as days off (1-in-7 rule)
- Reduces weekly hours
- Prevents overwork

**Potential issues:**
- Too many people out = coverage gaps
- Remaining residents may be overworked
- Supervision ratios may be harder to maintain

**Monitor:** Always check the Compliance page after adding absences or generating schedules.

## Exporting Absence Data

### Why Export?

- Share with administration
- Create reports
- Backup records
- Import into other systems
- Analysis and tracking

### Export Formats

#### CSV (Comma-Separated Values)
- Plain text format
- Opens in Excel, Google Sheets
- Good for spreadsheet analysis
- Compatible with most systems

#### JSON (JavaScript Object Notation)
- Structured data format
- For developers and system integration
- Contains full data structure

### How to Export

1. Navigate to the Absences page
2. Locate the **"Export"** button (near top-right)
3. Click the Export button
4. A dropdown menu appears
5. Select your desired format:
   - "Export as CSV"
   - "Export as JSON"
6. File downloads automatically
7. Open or save as needed

### Export Contents

The export includes:
- Person name
- Absence type
- Start date
- End date
- Notes
- All current absences (based on your filter)

**Note:** Export respects your current filters. If you've filtered to "Vacation only," only vacation absences will be exported.

### Using Exported Data

**In Excel/Sheets:**
- Sort by date to see chronological order
- Filter by person to see individual records
- Create charts and graphs
- Calculate total days by type
- Generate reports for administration

**For Integration:**
- Use JSON format for system integration
- Import into other scheduling systems
- Feed into reporting tools
- Archive for records

## Best Practices

### Requesting Time Off

**Do's:**
- Request as far in advance as possible
- Check rotation schedules before requesting
- Coordinate with fellow residents
- Provide detailed notes when helpful
- Follow your program's policies
- Keep track of your vacation balance

**Don'ts:**
- Don't wait until the last minute
- Don't request during critical rotation periods without discussion
- Don't assume approval - confirm first
- Don't forget to notify your team
- Don't request overlapping absences without checking coverage

### For Coordinators

**Daily:**
- Review new absence requests
- Check upcoming absences
- Monitor coverage gaps

**Weekly:**
- Review next 2-4 weeks of absences
- Identify potential conflicts
- Communicate with affected parties
- Adjust schedules if needed

**Monthly:**
- Generate upcoming month schedules with current absences
- Review vacation balance for all residents
- Plan for high-absence periods
- Export data for records

### Communication

**When requesting time off:**
1. Submit through the system
2. Also notify your immediate supervisor
3. Inform team members who might be affected
4. Confirm approval before making final plans
5. Arrange coverage for critical responsibilities

**When approving/denying:**
- Respond promptly to requests
- Explain reasons for denials
- Suggest alternative dates if needed
- Document decision in notes field

### Planning Ahead

**For residents:**
- Plan major vacations early
- Coordinate with co-residents
- Consider academic calendar
- Block time for conferences early
- Save some time for emergencies

**For programs:**
- Set vacation request deadlines
- Establish blackout periods clearly
- Create fair rotation policies
- Limit concurrent absences
- Plan for predictable patterns (conference season, holidays)

### Vacation Tracking

Although the system doesn't automatically track vacation balances:

**Recommendations:**
1. Export absence data regularly
2. Calculate vacation days used per person
3. Monitor against allocated time
4. Communicate remaining balance to residents
5. Track across academic year

**Create a tracking system:**
- Spreadsheet with vacation allocations
- Monthly review of usage
- Alert residents of remaining time
- Plan for use-it-or-lose-it scenarios

## Troubleshooting

### "I can't add an absence"

**Possible causes:**
- Missing required fields
- Invalid date range
- End date before start date
- Permission issues

**Solutions:**
- Check all fields are filled
- Verify dates are correct
- Ensure end date is after start date
- Contact admin if you lack permissions

### "My absence isn't showing in the calendar"

**Possible causes:**
- Date range is outside visible month
- Type filter is hiding it
- Page needs refresh

**Solutions:**
- Navigate to the correct month
- Check and adjust type filter
- Refresh the page (F5)
- Switch between List and Calendar view

### "Schedule still shows me working during my absence"

**Possible causes:**
- Absence added after schedule generation
- Wrong dates entered
- Schedule not regenerated

**Solutions:**
- Verify absence dates are correct
- Regenerate the schedule
- Contact coordinator for manual adjustment

### "Can't delete an absence"

**Possible causes:**
- Lack of permissions
- System error
- Already deleted but page not refreshed

**Solutions:**
- Verify you have edit permissions
- Refresh the page
- Contact admin if you should have access

## Advanced Tips

### Bulk Planning

For entering multiple absences:
1. Gather all dates first
2. Enter them in chronological order
3. Use copy-paste for repetitive notes
4. Double-check dates before saving

### Absence Conflicts

If multiple people need the same dates off:
1. Review coverage requirements
2. Assess priority (first-come, rotation importance, personal situations)
3. Negotiate and find alternatives
4. Document decisions
5. Communicate to all affected parties

### Year-End Planning

At academic year transitions:
1. Review remaining vacation balances
2. Encourage use of expiring time
3. Plan for July transition period
4. Enter known new-year absences early

### Integration with Other Systems

If your organization uses other systems:
1. Export absences regularly
2. Import into external calendar
3. Sync with HR systems if needed
4. Maintain system of record

## Frequently Asked Questions

**Q: How much vacation time do I get?**
A: Check with your program director. This varies by program and is not automatically tracked in this system.

**Q: Do I need approval before entering an absence?**
A: Depends on your program's policy. Some programs want you to get approval first, others use the system as the request mechanism.

**Q: What if I need emergency time off?**
A: Enter it as "Family Emergency" type and notify your supervisor immediately. Follow your program's emergency absence procedures.

**Q: Can I request partial days?**
A: Currently, the system tracks full days only. Coordinate partial days directly with your supervisor.

**Q: What happens if two people are out the same day?**
A: The system allows this but will account for it during schedule generation. It may create coverage gaps that need attention.

**Q: Can I see my vacation balance?**
A: The system doesn't currently calculate balances automatically. Export your absences and calculate, or ask your coordinator.

**Q: How far in advance can I request time off?**
A: You can enter absences for any future date. Follow your program's policies for how far ahead to request.

**Q: What if my plans change after approval?**
A: Edit or delete the absence as soon as possible and notify relevant parties.
