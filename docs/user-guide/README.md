## Quick Start

1. **Log in** with your credentials at the application URL
2. **Explore the Dashboard** to see schedule summary and compliance status
3. **Add People** - Set up residents and faculty
4. **Create Templates** - Define rotation types
5. **Record Absences** - Enter known time off
6. **Generate Schedule** - Create assignments automatically
7. **Check Compliance** - Verify ACGME requirements are met
8. **Export** - Download for distribution

## Key Features at a Glance

### ACGME Compliance Monitoring
- 80-hour weekly limit enforcement
- 1-in-7 day off requirement
- PGY-level supervision ratios
- Real-time violation tracking

### Smart Schedule Generation
- Multiple algorithm options
- Constraint-based optimization
- Absence-aware scheduling
- Faculty supervision assignment

### Schedule Resilience
- Faculty shortage planning
- Contingency schedule activation
- Workload and stress monitoring
- Load shedding priority guidance

### Military Support
- Deployment tracking
- TDY (Temporary Duty) management
- Integration with schedule generation

### Data Export
- Excel spreadsheets with color coding
- CSV for data analysis
- JSON for integrations
