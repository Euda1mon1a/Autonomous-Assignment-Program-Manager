# Residency Scheduler User Guide

Welcome to the Residency Scheduler user documentation! This guide will help you navigate and use the application effectively to manage residency rotations, absences, and ensure ACGME compliance.

## What is Residency Scheduler?

Residency Scheduler is a web-based application designed to help medical residency programs manage rotation schedules, track resident assignments, monitor absences, and ensure compliance with ACGME (Accreditation Council for Graduate Medical Education) requirements.

## Who This Guide Is For

This user guide is written for:

- **Program Coordinators** - Staff who manage daily scheduling operations, add residents, track absences, and generate schedules
- **Residents** - Medical residents who need to view their schedules, request time off, and check their assignments
- **Program Directors** - Faculty who oversee the program and need to review schedules and compliance
- **Administrative Staff** - Anyone involved in residency program management

## Key Features at a Glance

### Schedule Management
- Visual calendar showing resident assignments by day (AM/PM sessions)
- Automated schedule generation using constraint solving algorithms
- Block-based scheduling (typically 4-week periods)
- Real-time ACGME compliance checking

### Absence Tracking
- Calendar view of all absences
- Multiple absence types (vacation, medical, deployment, TDY, conference)
- Easy time-off request workflow
- Integration with schedule generation

### Compliance Monitoring
- Automatic validation of ACGME rules
- 80-hour work week tracking
- 1-in-7 day off rule enforcement
- Supervision ratio monitoring
- Detailed violation reports

### People Management
- Resident and faculty profiles
- PGY level tracking
- Specialty and qualification management
- Filtering by role and training level

### Export Capabilities
- Excel spreadsheet exports
- CSV data exports
- JSON format for integration
- Print-friendly views

## Common Tasks Quick Reference

| Task | Where to Go |
|------|-------------|
| View my schedule | Dashboard or Schedule page |
| Request time off | Absences → Add Absence |
| Generate a new schedule | Dashboard → Generate Schedule button |
| Check compliance status | Compliance page |
| Export schedule to Excel | Dashboard → Export Excel |
| Add a new resident | People → Add Person |
| View upcoming absences | Dashboard (Upcoming Absences card) |
| Change system settings | Settings (Admin only) |

## User Roles and Permissions

The application has different user roles with varying levels of access:

### Admin
- Full access to all features
- Can change system settings
- Can manage all users
- Can generate and edit schedules

### Coordinator
- Manage schedules, people, and absences
- Generate schedules
- View compliance reports
- Cannot access Settings page

### Faculty
- View-only access to schedules
- View compliance reports
- View people and absences
- Cannot edit or generate schedules
