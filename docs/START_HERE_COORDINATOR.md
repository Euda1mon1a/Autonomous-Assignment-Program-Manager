# Welcome, Program Coordinator!

> Your starting point for managing residency schedules

This guide helps you get productive quickly with the Residency Scheduler system.

---

## Your First 30 Minutes

1. **Login** - Navigate to the application URL provided by IT
2. **Dashboard Tour** - [Dashboard Guide](user-guide/dashboard.md)
3. **Daily Manifest** - See who is where today

---

## Your Daily Tasks

| Task | How To | Time |
|------|--------|------|
| Check today's coverage | Dashboard → Daily Manifest | 2 min |
| Review compliance alerts | Dashboard → Compliance Panel | 5 min |
| Process swap requests | Swaps → Pending | 10 min |
| Handle absence notifications | Absences → New Requests | 5 min |

### Daily Checklist

- [ ] Review Daily Manifest for coverage gaps
- [ ] Check for compliance warnings (yellow/red indicators)
- [ ] Process any pending swap requests
- [ ] Review new absence requests
- [ ] Address any flagged conflicts

## Key Features You'll Use Daily

### Dashboard
Your home base - shows coverage status, compliance alerts, and quick actions.
- **Location**: Home page after login
- **Guide**: [Dashboard](user-guide/dashboard.md)

### Daily Manifest
Real-time view of who is where today.
- **Location**: Dashboard → Daily Manifest
- **Guide**: [Daily Manifest](user-guide/daily-manifest.md)

### Swap Marketplace
Process shift swap requests from residents.
- **Location**: Navigation → Swaps
- **Guide**: [Swaps](user-guide/swaps.md)

### Compliance Monitor
Track ACGME work hour compliance.
- **Location**: Dashboard → Compliance
- **Guide**: [Compliance](user-guide/compliance.md)

## Common Questions

### "How do I see who's on call tonight?"
Dashboard → Call Roster, or [Call Roster Guide](user-guide/call-roster.md)

### "How do I approve a swap request?"
Swaps → Pending → Review → Approve/Deny, or [Swap Processing Playbook](playbooks/SWAP_PROCESSING_PLAYBOOK.md)

### "How do I generate next month's schedule?"
[Schedule Generation Playbook](playbooks/SCHEDULE_GENERATION_PLAYBOOK.md)

### "How do I handle a last-minute absence?"
[Absence Management Guide](user-guide/absences.md)

### "What do the compliance colors mean?"
- **Green**: Compliant
- **Yellow**: Warning (approaching limit)
- **Red**: Violation (requires immediate action)
