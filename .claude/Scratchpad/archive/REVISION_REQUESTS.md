## Request Template

When creating a new revision request:

```markdown
### REV-YYYY-NNN: [Brief Description]

**File:** `/path/to/file.md`
**Section:** [Specific section or "Entire file"]
**Priority:** P1 (Urgent) | P2 (High) | P3 (Medium) | P4 (Low)

**Issue:**
[What's wrong with the current content]

**Evidence:**
[How was this discovered - staleness scan, code change, user report]

**Suggested Direction:**
[What should the revision accomplish]

**Assigned To:** [Agent/Coordinator]
**Due Date:** YYYY-MM-DD
**Status:** Open | In Progress | Review | Closed

**Resolution:**
[Filled in when closed]
```
