Invoke the security-audit skill for security review.

## Arguments

- `$ARGUMENTS` - File path or "full" for complete audit

## Audit Areas

1. Authentication/authorization flaws
2. Data handling (PHI, PII)
3. HIPAA compliance
4. OPSEC/PERSEC violations
5. Injection vulnerabilities (SQL, XSS)
6. Secret management

## Output

Security report with severity ratings (Critical/High/Medium/Low).
