Invoke the docker-containerization skill for Docker help.

## Arguments

- `$ARGUMENTS` - Issue description or "status" for current state

## Capabilities

1. Debug container startup failures
2. Optimize Dockerfile for smaller images
3. Fix docker-compose configuration
4. Troubleshoot networking issues
5. Security scan containers

## Common Commands

```bash
docker compose ps
docker compose logs [service]
docker compose build --no-cache
```
